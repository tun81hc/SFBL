Release Notes for STM32L476G_EVAL  Board Drivers

Copyright 2015 STMicroelectronics

 

Update History

V0.1.0 / 14-January-2015

Main Changes

  • Beta release of STM32L476G_EVAL board revA (MB1144-A01) and revB (MB1144-B) drivers for STM32Cube L4 FW package:
    • revA / revB differences implemented under USE_STM32L476G_EVAL_REVA (default) and USE_STM32L476G_EVAL_REVB
      • LED3 moved from IO Expander pin IO1_PIN_3 to MCU pin PC1 on revB
      • PG10 used for FMC on revB
  • BSP drivers added : 
    • QSPI: stm32l476g_eval_qspi.h/.c
  • stm32l476g_eval.h
    • I2C_SPEED renamed to BSP_I2C_SPEED
  • stm32l476g_eval_audio.h/.c
    • Added defines for Audio output modes and Audio sample rates
    • New API BSP_AUDIO_OUT_ChangeAudioConfig() to change the audio output mode
    • Audio recording validated thru MP34DT01TR mems.
  • stm32l476g_eval_glass_lcd.h/.c
    • New API BSP_LCD_GLASS_BlinkConfig()
  • stm32l476g_eval_nor.h
    • BSP_NOR_xxxx() API status values NOR_OK/NOR_ERROR renamed to respectively NOR_STATUS_OK/NOR_STATUS_ERROR

V0.0.4 / 28-November-2014

Main Changes

  • Fourth Alpha release of STM32L476G_EVAL board RevA drivers (MB1144-A01) for STM32Cube L4 FW package
  • BSP drivers added : 
    • AUDIO: stm32l476g_eval_audio.h/.c
    • EEPROM: stm32l476g_eval_eeprom.h/.c
    • LCD TFT: stm32l476g_eval_lcd.h/.c
    • NOR Flash: stm32l476g_eval_nor.h/.c
    • SD: stm32l476g_eval_sd.h/.c
    • SRAM: stm32l476g_eval_sram.h/.c
    • TouchScreen: stm32l476g_eval_ts.h/.c
  • stm32l476g_eval.h/.c
    • Resources not mapped to MCU IO pins but via IO Expanders are put under USE_IOEXPANDER compilation switch: this concerns Joystick pins SEL and RIGHT/LEFT/UP/DOWN as well as LED2, LED3 and LED4
  • stm32l476g_eval_io.h/.c
    • New IO pin definitions for IO from IO Expanders: IO1_PIN_0 to IO1_PIN_7 and IO2_PIN_0 to IO2_PIN_15
    • APIs BSP_IOExpander1_xxx() and BSP_IOExpander2_xxx() changed to BSP_IO_xxx()

Known Limitations

  • The following board drivers are not yet managed
    • External memories interface via QuadSPI

V0.0.3 / 06-October-2014

Main Changes

  • Third Alpha release of STM32L476G_EVAL board RevA drivers (MB1144-A01) for STM32Cube L4 FW package
  • LCD glass BSP driver added : stm32l476g_eval_glass_lcd.h/.c
  • stm32l476g_eval.h
    • Align to STM32L4xx HAL Driver V0.1.0 for __HAL_RCC_XXX_CLK_ENABLE() / __HAL_RCC_XXX_CLK_DISABLE()

Known Limitations

  • The following board drivers are not yet managed
    • LCD TFT, EEPROM, SD, Audio and external memories interface via QuadSPI

V0.0.2 / 01-August-2014

Main Changes

  • Second Alpha release of STM32L476G_EVAL board RevA drivers (MB1144-A01) for STM32Cube L4 FW package
  • Cleanup done until SPI interface is integrated in STM32L476G_EVAL BSP board RevA drivers.

Known Limitations

  • The following board drivers are not yet managed
    • LCD, EEPROM, SD, Audio and external memories interface via QuadSPI

V0.0.1 / 11-July-2014

Main Changes

  • First Alpha release of STM32L476G_EVAL board RevA drivers (MB1144-A01) for STM32Cube L4 FW package
  • The following BSP drivers are supported:
    • LEDs
      • LED1 via GPIO: stm32l476g_eval.h/.c
      • LED2/LED3/LED4 via IO expander component STMPE811 over I2C: stm32l476g_eval.h/.c and stm32l476g_eval_io.h/.c
    • TAMPER push-button: stm32l476g_eval.h/.c
    • Joystick via IO expander component STMPE1600 over I2C: stm32l476g_eval.h/.c and stm32l476g_eval_io.h/.c
    • COM (USART1): stm32l476g_eval.h/.c

Known Limitations

  • The following board drivers are not yet managed
    • LCD, EEPROM, SD, Audio and external memories interface via QuadSPI
